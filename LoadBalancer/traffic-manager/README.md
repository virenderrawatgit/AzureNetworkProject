# Azure Traffic Manager

**Azure Traffic Manager** is a **DNS-based traffic routing service** that directs client requests to the most appropriate endpoint.

---

## Routing Methods

- Priority
- Weighted
- Performance
- Geographic
- Multi-value
- Subnet

---

## Key Characteristics

- DNS-level routing
- No data-plane traffic through Azure
- Endpoint health monitoring

---

## Use Cases

- Global failover
- Disaster recovery
- Multi-region applications
- Hybrid endpoints
