# Azure Load Balancer

**Azure Load Balancer** is a **Layer 4 (TCP/UDP)** load-balancing service that distributes traffic across backend resources within a region.

---

## Types of Azure Load Balancer

### 1. Public Load Balancer
- Internet-facing
- Used for public services
- Supports inbound and outbound traffic

### 2. Internal Load Balancer
- Private IP only
- Used for internal or backend services
- Common in multi-tier architectures

---

## Key Features

- High availability
- Health probes
- Inbound NAT rules
- Outbound SNAT
- Zone-redundant support

---

## Use Cases

- SQL Always On
- SAP
- Custom TCP/UDP applications
- Internal service load balancing
