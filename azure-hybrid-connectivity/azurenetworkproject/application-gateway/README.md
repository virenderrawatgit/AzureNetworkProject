# Azure Application Gateway

**Azure Application Gateway** is a **Layer 7 web traffic load balancer** with advanced routing, security, and SSL termination capabilities.

---

## Core Capabilities

- HTTP/HTTPS routing
- Path-based routing
- SSL termination
- Cookie-based session affinity
- Web Application Firewall (WAF)

---

## SKU Types

- Standard_v2
- WAF_v2

---

## Use Cases

- Web applications
- Microservices
- Secure ingress for AKS
- Multi-site hosting
