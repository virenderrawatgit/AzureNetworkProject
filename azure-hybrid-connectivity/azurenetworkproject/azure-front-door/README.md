# Azure Front Door

**Azure Front Door** is a **global Layer 7 load balancer** that operates at Microsoft’s edge locations.

---

## Key Features

- Global HTTP/HTTPS load balancing
- Anycast routing
- SSL offloading
- Web Application Firewall (WAF)
- URL-based routing

---

## Common Use Cases

- Global web applications
- Multi-region failover
- Performance optimization
- Internet-facing SaaS platforms
